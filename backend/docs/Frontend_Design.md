## Frontend Design Overview

The frontend of the AI-Based Study Planner is designed as a lightweight visualization layer that consumes AI-generated APIs from the backend. It does not contain business logic or analytical computations.

### Key Features
- Displays daily study plan
- Shows high-risk subjects
- Visualizes study time allocation
- Allows students to input academic data

### Technology Stack
- React.js
- Tailwind CSS
- Chart.js

### API Integration
Frontend interacts with backend through REST APIs, ensuring modularity and scalability.
