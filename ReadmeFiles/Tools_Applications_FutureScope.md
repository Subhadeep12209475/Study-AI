# Tools Used, Applications and Future Scope

## 1. Tools and Technologies Used

### 1.1 Development Tools
- Visual Studio Code – Code editor
- Git – Version control system
- Browser (Chrome/Firefox) – Application testing

---

### 1.2 Backend Technologies
- Python – Core programming language
- FastAPI – Backend framework for API development
- SQLAlchemy – ORM for database interaction

---

### 1.3 Frontend Technologies
- React.js – User interface development
- HTML, CSS, JavaScript – Web technologies
- Chart libraries – Data visualization

---

### 1.4 Database
- PostgreSQL – Relational database management system

---

## 2. Applications of the System
- Helps students plan daily study schedules effectively
- Assists in prioritizing subjects before examinations
- Improves time management skills
- Identifies academic risk areas at an early stage
- Can be used by college students and competitive exam aspirants

---

## 3. Advantages of the System
- Personalized study planning
- User-friendly interface
- Reduces academic stress
- Efficient utilization of study time
- Adaptable to different academic scenarios
